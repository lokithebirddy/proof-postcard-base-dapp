# Proof Postcard

## App Name

Proof Postcard

## Tagline

Mail proof on Base

## Description

Create a compact postcard proof with title, place, stamp, note, sender, and timestamp for events, launches, or milestones.

## Category

Proof / social / events

## Short Pitch

Proof Postcard turns a small real-world or digital moment into a clean Base postcard with a stamp, place, note, sender, and timestamp.

## Production Environment Variables

- `NEXT_PUBLIC_PROOF_POSTCARD_CONTRACT_ADDRESS`: deployed Proof Postcard contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
